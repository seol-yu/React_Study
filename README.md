yarn 설치함: `npm install yarn --global`

<br />

플젝 생성: `yarn create react-app react-project`

`cd react-project`

`yarn start`

<br />

크롬 웹 익스텐션 `React Developer Tools`

<br />

VSCode 익스텐션 `Reactjs code snippets(rcc enter 누르면 리액트 컴포넌트 생성)`, `Auto Import`
